# Steps

> 分步功能，结合路由的路径进行分步切换。

## 参数

| 参数  | 说明           | 类型  | 默认值 | 备注                      |
| ----- | -------------- | ----- | ------ | ------------------------- |
| items | 步骤条分多少步 | array | -      | ['step1','step2','step3'] |

## Demo

```html
<Steps :items="['step1','step2','step3']" />
```
